This is a package of example codes and data for the
**Optimization of hyperpolarized left ventricular [1-13C]pyruvate signal detection for human cardiac metabolic imaging**  
The main goal is to 
(i) define LV blood-pool and myocardial ROIs on a proton reference image, 
(ii) map those ROIs onto the 13C reconstruction grid, and 
(iii) quantify LV pyruvate signal distribution introducing an **exponential (B-value) model** and an **optimized threshold ROI Model** to select robust LV blood-pool voxels for downstream metabolic analysis (e.g., Lac+Bic in the mid-myocardium). **This code is applied to 106 HP studies**. 

Please download the .zip file, unzip it, and check the MATLAB code.
